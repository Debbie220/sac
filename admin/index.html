Controlling Presentations

  • Click on the Presentations tab to view all of the created presentations.
  • Click on a presentation's title to see more details about that presentation.
  • To edit a presentation, click the button.
  • The star indicates whether or not a presentation has been nominated for the Outstanding Undergraduate Research award.
  • The status of the presentation is shown highlighted in its corresponding color. “Saved” means the user has not yet submitted the presentation. “Pending Approval” means the user has submitted the presentation, but the admin has not yet approved it. “Approved” means that the presentation is approved, and is ready for scheduling. “Declined” means that there was something wrong with the presentation, and the admin has declined it. In this case, the user must make whatever changes are necessary, and submit it again.
  • To view all presentations that are “pending approval” and to approve/decline them, click in the top right corner of the page.
